Design Tokens

Scalable CSS custom properties, BEM naming conventions, fluid typography, and responsive grid patterns.

🎨

Design Tokens in :root

Define colors, fonts, radius, and shadows in :root to enforce site-wide visual consistency.

:root {
  --bg-primary: #050510;
  --accent-cyan: #06b6d4;
  --radius-md: 12px;
  --font-main: 'Outfit', sans-serif;
}
📐

Fluid Typography with clamp()

Use CSS clamp(min, val, max) for responsive font sizes without dozens of media queries.

h1 {
  font-size: clamp(2rem, 5vw, 4.5rem);
}
